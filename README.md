# darts-ranking

Minimal SolidStart hello-world scaffold with TypeScript 6, ESLint, Prettier, and plain imported CSS.

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run format:check
```

## Build

```bash
npm run build
npm run start
```

## Notes

- The app uses the stable SolidStart v1 starter.
- Styling is handled with normal imported CSS files under `src/`.
- ESLint currently relies on the TypeScript ESLint packages, whose peer dependency metadata still trails TypeScript 6 support.
